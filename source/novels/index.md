## 📋 使用說明

1. **新增小說**：在 `novels/` 資料夾創建新的 `.md` 文件
2. **更新列表**：複製上方模板，修改標題、檔名、時間和簡介
3. **本地預覽**：使用 `npm run server` 查看效果
4. **自動過濾**：部署時這些內容不會出現在公開網站

*🔒 注意：這些作品只會在本地預覽時顯示，不會出現在公開網站上。*
